# Sales-Dashboard-Project
Developed an interactive sales dashboard to analyze business performance across regions, categories, and customer segments. 
Key Features:  (i) KPI  (Total Sales, Profit, Quantity) 
(ii) Sales analysis by Region and Category 
(iii) Profit analysis by Segment 
(iv) Interactive filters (Region, Category, Segment)

Insights:
(i) Identified top-performing regions
(ii) Analyzed profitable vs loss-making segments
